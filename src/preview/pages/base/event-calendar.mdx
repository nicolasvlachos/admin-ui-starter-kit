<Section title="Overview" id="overview">

`EventCalendar` is the canonical month view — pass `events` and optional
`categories`, drive filtering with `enableCategoryFilter` / `visibleCategories`
or imperatively with `filterEvent`, and slot a custom renderer with
`renderEvent`. Navigation can be capped via `minDate` / `maxDate` and
`disabledDates`, and every string + locale is overridable.

</Section>
