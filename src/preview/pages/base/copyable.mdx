<Section title="Overview" id="overview">

`Copyable` wraps any value in a click-to-copy span. The copy icon appears on
hover, the value lands on the clipboard, and a `sonner` toast confirms (mount
a `<Toaster />` somewhere in the tree). Supports `mono`, `truncate`, custom
`displayValue`, and a custom `successMessage`.

</Section>
