<Section title="Overview" id="overview">

`MetadataList` renders an array of `{ id, label, value, icon?, tooltip?,
description?, render?, emptyLabel? }` items. List-level props pick the
`layout` (`vertical` / `rows` / `horizontal`), the `columns` count (1–4),
and density. Per-item `render` escapes the default formatting for status
pills, copyable IDs, links, or inline edits.
