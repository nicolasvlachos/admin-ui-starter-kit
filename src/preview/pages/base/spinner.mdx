<Section title="Overview" id="overview">

`Spinner` ships three variants — `default` (rotating SVG), `shimmer` (round
shimmer dot), `progress` (slim animated bar). The variant resolves through
`useSpinnerConfig().defaultVariant`, so a consumer can pin the look at the
`UIProvider` root and override per call. Pass an optional `label` for screen
readers and a visible status message.
