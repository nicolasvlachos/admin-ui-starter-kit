<Section title="Overview" id="overview">

`Item` is the row primitive used everywhere a "media + title-stack + actions"
shape recurs — admin rows, contact details, search results, settings toggles,
dropdown menus. Compose `ItemMedia`, `ItemContent` (with `ItemTitle` and
`ItemDescription`), and `ItemActions` inside an `Item`. Wrap multiple rows in
`ItemGroup` (with optional `ItemSeparator`s between them) for a list.

The `render` prop turns any `Item` into a custom element (anchor, button, link
wrapper) without breaking focus/hover styling.
