<Section title="Overview" id="overview">

`MoneyDisplay` renders a primary (and optional secondary) currency amount,
using the active `useCurrency()` config for formatting, dual-pricing, and
display mode. `CurrencyPairPreview` renders an inline `source → target`
conversion. Both honour `UIProvider` config so consumers can flip the entire
app to dual pricing or a different default currency in one place.
