<Section title="Overview" id="overview">

The map components are thin React wrappers around Leaflet. Always wrap a
`<Map>` in a parent with a fixed height (Leaflet renders absolute children).
Compose `<MapTileLayer />` for the basemap, `<MapMarker>` for a pin, and
`<MapPopup>` inside a marker for the click bubble.
